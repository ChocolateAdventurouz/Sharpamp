================================================================
Sharpamp 1.0 Beta - Create Winamp plugins in C#
By Daniel15 - http://www.d15.biz/
http://code.google.com/p/sharpamp/
================================================================

Sharpamp allows you to create Winamp plugins using C#. Please read the "Getting Started" guide available at http://code.google.com/p/sharpamp/ to see how to create a plugin.

Requirements:
=============
 - Visual Studio 2008 (should work in 2005 as well as 2010 beta, but this has not been tested yet)
   - C# and C++ both have to be installed
 - Winamp
